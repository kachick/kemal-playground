# kemal-playground

My sandbox for <https://github.com/kemalcr/kemal>

## Setup

Use direnv and Nix flake

```console
> shards install
Resolving dependencies
```

## Usage

```console
> just db-start
> just db-migrate
> just dev
[development] Kemal is ready to lead at http://127.0.0.1:3000
```

Go to <http://127.0.0.1:3000>
