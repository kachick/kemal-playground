# kemal-playground

This repository was for learning Kemal basics. I no longer maintain this project.

## Setup

Use direnv and Nix flake

```console
> shards install
Resolving dependencies
```

## Usage

```console
> just db-start
> just db-migrate
> just dev
[development] Kemal is ready to lead at http://127.0.0.1:3000
```

Go to <http://127.0.0.1:3000>
